# Consejo Asesor (ONFALO)

Plugin para Scriptorium ALEPH que integra el sistema de consejo-asesor de ONFALO: 14 agentes intelectuales en 7 modos dialecticos con pipeline de produccion de relatos.

## Skills

| Skill | Descripcion |
|-------|-------------|
| `/debate` | Debate dialectico: protagonista vs antagonista(s) |
| `/auditoria` | Auditoria triple: verdad + sombra + registro |
| `/produccion` | Producir articulo o integrar material en capitulo |
| `/corresponsalia` | Periodismo de doble realidad (ecumenico/magico/cruce) |
| `/crear-proyecto` | Crear proyecto ONFALO-compatible con proyecto.config.md |
| `/pipeline-relato` | Pipeline 3 fases: laboratorio -> fichas -> relato |

## Datos

- **Plugin (infra)**: `ARCHIVO/PLUGINS/CONSEJO_ASESOR/`
- **Usuario (proyectos)**: `ARCHIVO/DISCO/CONSEJO/{nombre}/`
