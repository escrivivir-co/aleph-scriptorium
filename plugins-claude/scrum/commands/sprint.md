# Sprint Ceremony

Run a Scrum sprint ceremony.

Ceremony types:
- **planning** - Plan the next sprint from backlog
- **review** - Review completed work in current sprint
- **retro** - Run a retrospective
- **close** - Close and archive current sprint

If no ceremony specified, show current sprint status and ask which ceremony to run.
