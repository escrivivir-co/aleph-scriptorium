## 🎬 Demo Rápida para el Público

### 1️⃣ Ver Agentes en Acción

```bash
@aleph —> redacta contenido
         ├→ @revisor verifica coherencia
         └→ @periodico publica noticia
```

### 2️⃣ Auditoría de Verdad (5 Banderas)

Contenido validado contra:
- 🔵 **Blueflag**: Evidencia, utilidad, falsificabilidad
- ⚫ **Blackflag**: Autodefensa, represión
- 🔴 **Redflag**: Escala, enforcement
- 🟡 **Yellowflag**: Límites, inconmensurabilidad
- 🟠 **Orangeflag**: Registro, género, estilo

### 3️⃣ Estado DevOps en Tiempo Real

```bash
curl http://localhost:3003/health  # ✅ HEALTHY
curl http://localhost:4001/status   # ✅ READY
```
