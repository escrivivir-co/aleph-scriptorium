# Escena 2: Tirada de Cartas

> **Estadio**: 2/3  
> **Anillo**: 1 (Primera expansión)  
> **Tipo**: prueba

---

## Ambientación

*El Tarotista baraja el mazo con movimientos precisos. Las cartas brillan con un fulgor tenue.*

---

## Prueba Interactiva

### Tu tarea

Piensa en una afirmación que quieras evaluar. Puede ser:

- Una propuesta política o social
- Una teoría que has escuchado
- Una afirmación de un texto que estás leyendo
- Una intuición que tienes pero no sabes si es fundada

**Ejemplo**: *"La inteligencia artificial puede reemplazar completamente el trabajo creativo humano."*

---

## El Tarotista responde

*Cuando el usuario presenta su afirmación, el Tarotista realiza la tirada.*

> Bien. Veo tu propuesta.
> 
> Antes de juzgarla, necesito entender qué tipo de afirmación es.

*Coloca cuatro cartas en forma de cruz.*

### Los Cuatro Cuadrantes

```
                    INTERIOR
                       │
        ┌──────────────┼──────────────┐
        │              │              │
        │   COPAS      │   ESPADAS    │
        │  (Sentido)   │  (Verdad)    │
        │              │              │
INDIV.──┼──────────────┼──────────────┼──COLECT.
        │              │              │
        │   BASTOS     │    OROS      │
        │  (Acción)    │  (Materia)   │
        │              │              │
        └──────────────┼──────────────┘
                       │
                   EXTERIOR
```

> - **Copas** (Interior-Individual): ¿Qué significa para ti personalmente?
> - **Espadas** (Interior-Colectivo): ¿En qué paradigma cultural se inscribe?
> - **Bastos** (Exterior-Individual): ¿Qué comportamientos observables predice?
> - **Oros** (Exterior-Colectivo): ¿Qué condiciones materiales lo harían verdadero o falso?

*El Tarotista señala la carta de Oros.*

> La pregunta de demarcación vive principalmente aquí: en los **Oros**.
> 
> Popper preguntaría: ¿Qué evidencia empírica refutaría esta afirmación?
> 
> Pero no cometas el error de pensar que solo los Oros importan. 
> Las Copas (sentido) tienen dignidad propia, aunque no sean "científicas".
