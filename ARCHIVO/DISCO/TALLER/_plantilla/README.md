# Plantilla de Proyecto Teatral

> Copia esta carpeta para iniciar un nuevo proyecto de obra.

## Instrucciones

1. Copia la carpeta `_plantilla/` con el nombre de tu proyecto (kebab-case)
2. Edita `obra.yaml` con tu trama de 12 estadios
3. Crea personajes en `personajes/`
4. Desarrolla escenas en `escenas/`
5. Invoca `@teatro` para instalar

## Estructura

```
{tu-proyecto}/
├── README.md           ← Estás aquí
├── obra.yaml           ← Define los 12 estadios
├── personajes/         ← Un YAML por personaje
│   └── protagonista.yaml
├── escenas/            ← Un MD por escena
│   └── 01-inicio.md
└── assets/             ← Imágenes, etc.
    └── poster.png
```
