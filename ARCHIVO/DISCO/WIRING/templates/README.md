# Plantillas Node-RED

> **Plugin**: WireEditor

---

## Descripción

Esta carpeta contiene **plantillas reutilizables** de flows, subflows y nodos.

---

## Estructura

```
templates/
├── README.md
├── flows/           # Flows completos
├── subflows/        # Subflows reutilizables
└── nodes/           # Definiciones de nodos custom
```

---

## Fuente de Plantillas

Las plantillas vienen de:
- **Submódulo**: `node-red-alephscript-sdk/examples/`
- **Proyectos**: Flows probados que se convierten en plantillas
- **Comunidad**: Importados de otros repos

---

## Sincronizar Plantillas

Para traer plantillas del submódulo:

```
@wireeditor sincronizar con submódulo
```
