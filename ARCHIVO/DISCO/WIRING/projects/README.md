# Proyectos Node-RED

> **Plugin**: WireEditor

---

## Descripción

Esta carpeta contiene **proyectos Node-RED** creados para casos de uso específicos.

---

## Estructura de un Proyecto

```
{nombre}/
├── flow.json          # Flow principal
├── package.json       # Dependencias (opcional)
├── README.md          # Documentación
└── .node-red/         # Config local (opcional)
```

---

## Crear Proyecto

```
@wireeditor crear proyecto {nombre}
```

El agente:
1. Crea la carpeta
2. Genera flow.json inicial
3. Crea README.md
4. Actualiza catalog.json

---
